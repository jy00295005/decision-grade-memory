# Section Drafting Prompt Template

## Task

Draft or revise one manuscript section only.

## Target file

`manuscript/sections/TODO_section_file.md`

## Inputs

- Current section file
- `README.md`
- `docs/outline.md`
- `docs/rq.md`
- Relevant notes or real sources only

## Constraints

- Work only on the target section.
- Preserve academic tone and precise wording.
- Do not invent citations, results, or claims.
- Use `TODO:` for missing evidence or unresolved points.
- Prefer a small, traceable edit over a broad rewrite.

## Desired output

- Updated section draft in place
- Brief note on what changed
- One actionable follow-up item if a clear next step appears

## Verification checklist

- Is the section aligned with the outline?
- Are unsupported claims marked with `TODO:`?
- Is terminology consistent with the rest of the repo?
- Did the edit avoid changing unrelated files?
