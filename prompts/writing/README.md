# Writing Prompt Pack

This directory contains reusable prompts for manuscript drafting and revision at the section level.

## Files

- `section_drafting.md`
  - generic template for drafting or revising one section at a time

## Use guidance

- Use this pack when the chapter or section structure is already clear.
- Prefer this after literature and positioning work are stable enough to support paper prose.
- Keep edits local, traceable, and section-specific.
