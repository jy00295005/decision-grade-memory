# Literature Review Prompt Pack

This directory stores the staged prompts used for the literature review and research-positioning workflow in this repository.

## Recommended order

1. `01_round1_reconnaissance.md`
2. `02_sme_decision_pain_points.md`
3. `03_ai_limits_for_decision_support.md`
4. `04_research_positioning.md`

## What each prompt does

- `01_round1_reconnaissance.md`
  - broad first-pass reconnaissance on SME decision pain points and AI support limits
- `02_sme_decision_pain_points.md`
  - narrows to recurring SME decision difficulties in practice
- `03_ai_limits_for_decision_support.md`
  - investigates what current AI support systems do and do not reconstruct well
- `04_research_positioning.md`
  - synthesizes findings into a narrower paper gap and positioning note

## Use guidance

- Use these prompts before serious chapter drafting.
- Keep claims conservative and source-backed.
- Treat outputs as research notes first, not as finished paper prose.
