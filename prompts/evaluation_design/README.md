# Evaluation Design Prompt Pack

This directory stores the prompts used to design and then export Section 4 / Evaluation Design.

## Recommended order

1. `01_section4_round1_design.md`
2. `02_section4_chapter_export.md`

## What each prompt does

- `01_section4_round1_design.md`
  - creates modular evaluation-design files before paper-style drafting
- `02_section4_chapter_export.md`
  - assembles those files into a coherent chapter and exports a PDF

## Use guidance

- Use this pack after the Method section is stable enough to define an experimental comparison.
- Keep the empirical framing narrow:
  - context construction quality
  - not broad product benchmarking
