Create an academic conceptual framework diagram for a paper on scenario memory for SME decision support.

5. LLM Reasoning Layer
   - explanation

provenance, recency, confidence, evidence linkage.

Title:
"A Scenario Memory Framework for SME Decision Support"

Make it look like a polished research figure, not a corporate marketing slide.

Additional requirements:
- balanced spacing between blocks
- consistent visual hierarchy
- arrows should make the left-to-right flow obvious
- labels should remain legible at paper size
- preserve a white-background academic figure style
