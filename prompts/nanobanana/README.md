# Nanobanana Figure Prompts

This directory stores reusable image-generation prompts for the key figures used in Section 3 / Method.

Target paper:

`Bridging Data, Signals, and Growth: A Graph-Based AI for SME Decision-Making`

## Purpose

These prompts are written for academic figure generation, not for marketing visuals. They are intended to help reproduce or refine publication-friendly diagrams for the Method section.

## Files

- `fig1_fragment_vs_scenario.md`
  - motivation figure comparing fragmented retrieval, scenario memory, and evidence-backed reasoning
- `fig2_framework_overview.md`
  - overall framework figure for the method section
- `fig3_decision_episode_structure.md`
  - semantic structure figure for the decision episode representation

## Use guidance

- Keep the figures academic, white-background, and publication-friendly.
- Prefer legibility at paper size over decorative complexity.
- Avoid corporate slide aesthetics, poster-like styling, or cartoonish sketch-note effects.
- If a generated image is too polished in a commercial sense, pull it back toward research-figure clarity.
- If a generated image is too rough or sketchy, tighten alignment, spacing, and label hierarchy.
