# Chapter Workflow Template

## Inputs

- chapter source files
- optional figures
- optional figure captions
- optional audit or reference notes

## Assembly steps

1. Read all source files.
2. Identify the strongest chapter structure.
3. Mark repeated claims, definitions, and transitions.
4. Merge source material into one chapter working draft.
5. Rewrite into cohesive academic prose.
6. Insert figure placeholders or references where useful.
7. Create export-ready Markdown.
8. Generate chapter PDF.

## Output split

- working draft
- polished export draft
- optional HTML export
- final PDF
