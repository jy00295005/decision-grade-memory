# External Skills Manifest

This file lists the non-`dgm` skills that are part of the paper workflow for this repository.

These skills are used locally in Codex, but they are not authored in this repository.

## Research and writing skills used in this project

- `literature-review`
  - structured literature reconnaissance and synthesis
- `bgpt-paper-search`
  - paper discovery and evidence expansion
- `citation-management`
  - citation verification and bibliography hygiene
- `markitdown`
  - source file to Markdown conversion
- `markdown-mermaid-writing`
  - Markdown-first academic writing and diagram support
- `hypothesis-generation`
  - framing hypotheses and competing explanations
- `exploratory-data-analysis`
  - empirical artifact inspection and pilot-data analysis
- `networkx`
  - graph-oriented modeling and structure support

## Notes

- These skills were installed locally for this project workflow.
- Their original installation source should be documented separately when reproducing the environment.
- The repo-tracked custom workflow skills are the `dgm-*` skills under `codex/paper_skills/custom/`.
